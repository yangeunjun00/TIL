# 구조 분해 할당

### JavaScript

````
const arr = [1,2,3,4];
const [x,y,z] = arr;
console.log(x) // 1
console.log(y) // 2
console.log(z) // 3```


````

const obj = {
a: "hi"
b: 7
};
const {a,b} = obj
console.log(a) // hi
console.log(b) // 7```

### React

코드로 쓰기 귀찮으니까 말로 대충 설명 하도록 하겠다.

App 컴포넌트와 자식 컴포넌트 zasik 컴포넌트가 있다고 하자

App 컴포넌트에서

```
const obj = {
    name: "양은준"
    age: "18"
};
```

이 obj 객체를 props로 zasik 컴포넌트에게 넘겨준다고 하자

일반 적으로 넘겨주면

```
// App
return(
    <div>
        <zasik a = {obj}>
    </div>
)
```

```
// zasik

const a = props =>{
return (
<div>
<span>{props.obj.name}</span>
<span>{props.obj.age}</span>
</div>
)
}
```

이렇게 props.obj.name , props.obj.age 처럼 데이터 하나 처리하는데 몇번의 타자를 쳐야하는건지 정말 너무나도 길다
이 수고를 덜기 위해 구조 분해 할당을 하면

```
// App 은 위와 동일


//zasik


const a = props =>{
    const {name,age} = props;
    return (
<div>
<span>{name}</span>
<span>{age}</span>
</div>
)
}
```

이렇게 그냥 name , age 만 쓰면 된다.
