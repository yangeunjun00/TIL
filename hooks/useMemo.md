# useMemo hook

useMemo를 소개 하기 전 *메모이제이션*에 대해 말해보겠다.

## 메모이제이션이란?

동일한 값을 리턴 하는 함수를 여러번 호출할 때

처음 값을 계산하고 메모리에 캐싱한 후

필요할 때 메모리에서 캐시해와서 값을 재사용하는 기법이다.

이 메모이제이션을 할 수 있게 하는 hook이 바로 useMemo이다.

### 사용법

```
const memo = useMemo(()=>{
    return memo = memo+1;
},[memo]);
```

이렇게 인자로 익명 함수를 쓰고 useEffect와 유사하게 의존성 배열에 넣은 변수가 업데이트 될 때만 실행된다.

이로 인해 메모리제이션 뿐만 아니라 컴포넌트 내부에서 다른 값이 변경되어도 다시 렌더링 되는 현상을 없앨 수 있다.
